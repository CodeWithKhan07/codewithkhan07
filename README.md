## 🧠 About

I build production Flutter apps end-to-end — architecture, implementation, and shipping to the App Store — on a stack built around **Riverpod**, clean architecture, and enough SOLID discipline to keep a codebase sane six months later.

At **Chimpgroup**, I work on **QDrive**, a vehicle rental platform where entire screens are defined as JSON on the backend and rendered dynamically — a "ship once" approach that skips App Store review for most UI changes. Before that, at **EmergeSol Solutions**, I was the sole developer on three shipped production apps.

- 🔭 Currently building **QDrive** — a server-driven UI engine for a mobility platform
- 🧩 Equally comfortable with Riverpod, GetX, and BLoC — I pick the tool the codebase needs
- 📦 Shipped apps covering real-time chat, offline-first storage, and API-driven content
- ⚡ Fun fact: my personal projects run on GetX because I like an idea working by the weekend, not the sprint
